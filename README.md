## Installation

Installation is super-easy via [Composer](https://getcomposer.org)

```bash
composer require filesys/path
```

or add it by hand to your `composer.json` file.

## Features

- separate
- basename
- dirname
- extname
- rootname
- filename
- pathname
- format
- parse
- isAbsolute
- lastDir
- join
- real
- BSEP
- FSEP
- sep
- isLocal
- delimiter
- normalize
- relative
- resolve
- filePaths
- toNamespacedPath
